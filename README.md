# Mauritius Travel Agency System

A comprehensive travel agency management system tailored for Mauritius.

## Project Structure

- `backend/`: Node.js/Express API (Shared)
- `web-app/`: Customer-facing React Application
- `admin-app/`: Admin Management React Application
- `mobile-app/`: Cross-platform React Native (Expo) App
- `docs/`: Project Documentation
- `scripts/`: Utility scripts for development and deployment
- `tests/`: Integration and E2E tests

## Technologies

- **Backend**: Node.js, Express, MongoDB, Amadeus API, Twilio, Nodemailer.
- **Frontend**: React.js, Vite, Tailwind CSS, Lucide React.
- **Mobile**: React Native, Expo, NativeWind.
- **DevOps**: Docker, AWS/Heroku.

## Compliance

- Mauritius Data Protection Act 2017
- GDPR Compliance (Minimal data collection, 12-month retention policy)

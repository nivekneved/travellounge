# Travel Lounge 2026 Web App

This is the front-end application for Travel Lounge, built with React and Vite.

## Features

- Showcases Flights, Hotels, Tours, Cruises, and Packages.
- Booking functionality with loading states and confirmation alerts.
- Premium UI/UX design.

## Development

1. Ensure Node.js is installed.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the development server.

## Structure

- `src/components`: UI components.
- `src/pages`: Main application pages.
- `src/context`: React contexts (e.g., Wishlist).
- `src/hooks`: Custom hooks.
- `src/utils`: Helper functions.
